본문 소개 이용방법 점자

+ 서비스 소개

세상만짐은 시각 장애인분들께서 사용하시는 정보 접근 기술의 발전을 위한 실시간 뉴스 점역 데이터 제공 서비스입니다.

네이버에서 제공하는 실시간 인기 뉴스를 테마(정치, 경제, 세계, 사회)에 따라 크롤링한 뒤,

저희가 고안한 json 구조(BraileNewsJSON)에 따라 크롤링한 데이터를 점역하여 제공합니다.

현재 beta 서비스로 간단한 점역 기능을 가지고 있습니다.

추후에 약어와 이미지에 대한 점자 등을 제공하여

시각 장애인분들을 위한 서비스를 구현하시는데 유용한 데이터를 제공하고자 합니다.

+ BraileNewsJSON 형식

BraileNewsJSON의 형식은 아래와 같습니다.

{
        "header": {
                "encoding": ""
                "title": "기사 제목"
                "topic": "기사 주제"
                "date": "기사 발행 일자"
                "url": "기사 주소"
                "crawling-function": "크롤링 프로그램 소스 주소"
                "translation-rules": "번역 규칙 테이블 소스 주소"
                "translation-function": "번역 프로그램 소스 주소"
        }
        "body": {
                "original-version": "원본 - 크롤링 데이터"
                "traslated-version": "점자 데이터"
        }
}

BraileNewsJSON의 형식의 예시입니다.

{
        "header": {
                "encoding": "UTF-8"
                "title": " MBN PICK 안내 [MBN 여론조사] 이재명 37.9% vs 윤석열 43.3%…윤 우세..."
                "topic": "Policy"
                "date": "2021.12.02"
                "url": "https://news.naver.com/main/read.naver?mode=LSD&mid=shm&..."
                "crawling-function": "https://github.com/TwoJJung/NewsForDeafBlind"

        }
}
